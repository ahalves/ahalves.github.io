## If (else) statements
```cpp
int main()
{
	int a = 1;
	int b = 2;

	if (a > b)
	{
		cout << "a is greater than b" << endl;
		return 0; 
	}
	else if (a < b)
	{
		cout << "a is less than b" << endl;
		return 0; 
	}
	else
	{
		cout << "a and b are equal" << endl;
		return 0; 
	}
}
```
